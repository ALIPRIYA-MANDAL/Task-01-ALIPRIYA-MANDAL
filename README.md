# Rule-Based AI Chatbot

## Description
A simple rule-based chatbot built using Python.

## Features
- Greets users
- Responds to predefined questions
- Uses if-else logic
- Runs continuously using a loop
- Handles exit commands
- Provides fallback responses

## Technologies Used
- Python

## How to Run
1. Open terminal
2. Run:
   python chatbot.py
